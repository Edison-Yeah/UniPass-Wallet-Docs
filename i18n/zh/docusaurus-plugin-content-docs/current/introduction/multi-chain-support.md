# 多链支持

UniPass Wallet 可以支持所有 EVM 兼容链，并通过 EIP-2470 保持多链同一钱包地址。目前支持的链如下：

### 主网

- BSC（Binance Smart Chain）
- Polygon
- Rangers protocol

### 测试网

- BSC testnet
- Mumbai（Polygon testnet）
- Robin（Rangers testnet）

## 未来将支持的链

- Ethereum
- Arbitrum
- Optimism
- Fantom
- Avalanche
